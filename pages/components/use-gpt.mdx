# useGPT

The `useGPT` hook allows you to call any methods supported by google publisher tag. It must be called from a component nested within a `GPTProvider`.

```tsx showLineNumbers
import { GPTSlot, useGPT } from 'goopubtag';

// Assuming `GPTProvider` defined higher up
const MyComponent = () => {
  const { refresh } = useGPT();
  const onClick = () => refresh(['foo']);
  return (
    <div>
      <p>Ad foo</p>
      <GPTSlot adUnit='abc/xyz' id='foo' />
      <button onClick={onClick}>Refresh foo</button>
      <p>Ad baz</p>
      <GPTSlot adUnit='abc/xyz' id='baz' />
    </div>
  );
};  

export default MyComponent;
```

Here is a complete list of methods returned from `useGPT`:

### refresh

- type: `(adSlotIds?: string[]) => void`

Calling `refresh` will refresh ads specified in the `adSlotIds` array provided. Calling `refresh` with no arguments will refresh all ads on the page.
